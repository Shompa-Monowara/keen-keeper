# 📌 KeenKeeper

## 📝 Short Description
**KeenKeeper** is a smart Personal Relationship Manager (PRM) built with React. It’s designed to help users maintain consistent connections with their friends through organized tracking, instant interaction logging, and data-driven analytics. With a clean and intuitive dashboard, KeenKeeper ensures you never lose touch with the people who matter most.

---

## 🛠️ Technologies Used
This project leverages a modern frontend stack for a fast and responsive user experience:

* **React.js** – Component-based UI development and state management.
* **Tailwind CSS & DaisyUI** – Modern styling and polished UI components for a professional look.
* **React Router DOM** – Seamless navigation between Home, Timeline, and Stats pages.
* **Recharts** – Data visualization for friendship interaction patterns.
* **React Icons** – Enhancing UX with intuitive iconography.

---

## 🚀 Key Features

### 1. Automated Connection Tracking
KeenKeeper intelligently tracks the **"Days Since Last Contact"** for each friend. It automatically categorizes contacts as **On-Track** or **Overdue** based on your communication frequency, helping you prioritize who to reach out to next.

### 2. Real-Time Interaction Timeline (Search & Sort)
A centralized timeline that logs every **Call, Text, and Video Chat**. It includes advanced **Search** (by friend name) and **Sort** (Newest/Oldest) functionalities, allowing you to easily browse through your social history and interaction types.

### 3. Visual Social Analytics
The dedicated **Stats Page** uses dynamic Pie Charts to visualize your interaction habits. It breaks down your communication by type (Call vs. Text vs. Video), giving you a clear picture of how you connect with your social circle.
