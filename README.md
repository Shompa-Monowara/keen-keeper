# 📌 KeenKeeper

## 📝 Short Description
**KeenKeeper** is a smart React-based dashboard designed to help you stay consistently connected with your close friends.  
It combines clean UI with tracking and analytics so you can easily manage your relationships and never lose touch with the people who matter most.

---

## ✨ Core Functionalities
- **Smart Friend Tracking:** Automatically organizes contacts into a responsive 4-column grid with real-time status labels (**Overdue**, **On-Track**) to highlight who needs attention.  
- **Seamless Interaction Logging:** Includes a **Quick Check-in** feature to log calls, texts, or video chats instantly, updating the interaction history in real time.  
- **Dynamic Timeline & Analytics:** Provides a filterable timeline of interactions and visualizes communication patterns using **Recharts Pie Charts**.  
- **Polished User Experience:** Built with **Tailwind CSS**, smooth navigation, loading states, and toast notifications for a professional app-like experience.  

---

## 🛠️ Technologies Used
This project was built using a modern frontend stack for speed and responsiveness:

- **React.js** — Component-based UI development  
- **React Router DOM** — Smooth page navigation (**Home**, **Timeline**, **Stats**)  
- **Tailwind CSS + DaisyUI** — Modern styling and UI components  
- **Recharts** — Interactive charts and friendship analytics  

---

## 🚀 Key Features
### 1. Automated Connection Tracking
Tracks **Days Since Contact** and automatically marks friends as **On-Track** or **Overdue** based on communication goals.

### 2. Instant Interaction Logging
Log **Call**, **Text**, or **Video Chat** interactions with one click.  
Updates the centralized timeline instantly.

### 3. Visual Social Analytics
A dedicated **Stats page** displays Pie Charts showing interaction patterns, making it easy to understand communication habits.
