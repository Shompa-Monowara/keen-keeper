# 📌 KeenKeeper

## 📝 Short Description
**KeenKeeper** is a smart Personal Relationship Manager (PRM) built with React. We designed it to help users maintain consistent connections with their friends through organized tracking, instant interaction logging, and data-driven analytics. With a clean and intuitive dashboard, KeenKeeper ensures we never lose touch with the people who matter most.

---

## 🛠️ Technologies Used
We leveraged a modern frontend stack to ensure a fast and responsive user experience:

* **React.js** – Component-based UI development and state management.
* **Tailwind CSS & DaisyUI** – Modern styling and polished UI components for a professional look.
* **React Router DOM** – Seamless navigation between Home, Timeline, and Stats pages.
* **Recharts** – Data visualization for friendship interaction patterns.
* **React Icons** – Enhancing UX with intuitive iconography.

---

## 🚀 Key Features

### 1. Automated Connection Tracking
KeenKeeper intelligently tracks the **"Days Since Last Contact"** for every friend in our list. We automatically categorize contacts as **On-Track** or **Overdue** based on communication frequency, helping us prioritize who to reach out to next.

### 2. Real-Time Interaction Timeline (Search, Filter & Sort)
We implemented a centralized timeline that logs every **Call, Text, and Video Chat**. It includes advanced **Search** (by friend name), **Category Filtering** (to isolate specific interaction types), and **Smart Sorting** (Newest/Oldest), allowing us to easily browse and manage our social history.

### 3. Visual Social Analytics
Our dedicated **Stats Page** uses dynamic Pie Charts to visualize interaction habits. We break down communication by type (Call vs. Text vs. Video), giving us a clear picture of how we connect with our social circle.
